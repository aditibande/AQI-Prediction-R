# AQI Prediction & Trend Analysis (MP Districts)

## 📊 Overview
This project analyzes historical AQI data (2020–2025) and predicts AQI levels for 2027 using linear regression.

## 🚀 Features
- Data cleaning from multi-year datasets
- Trend analysis of AQI across districts
- Prediction model for AQI (2027)
- Visualization using ggplot2
- Shiny dashboard for interactive analysis

## 🛠 Tech Stack
- R
- Tidyverse
- ggplot2
- Shiny

## 📈 Output
- District-wise AQI trends
- Top polluted districts prediction
- AQI forecasting model
